How to use:

1. launch iCEcube
2. open the rgbw_lamp_new_sbt.project in /rgbw_lamp_new/ folder
3. open Diamond Programmer and open the programmingData.xcf in the /rgbw_lamp_new folder/
4. open the rgbw_lamp_bitmap.bin in /rgbw_lamp_new/rgbw_lamp_new_Implmnt/sbt/outputs/bitmap/

If needed:

How to regenerate bit file:
1. launch iCEcube
2. open the rgbw_lamp_new_sbt.project in /rgbw_lamp_new/ folder
3. run through the flow to generate bit file
4. the generated bit files are under rgbw_lamp_new\rgbw_lamp_new\rgbw_lamp_new_Implmnt\sbt\outputs\bitmap
